Just copy/paste, it'll all be fine. sudo apt-get install collectd --yes sudo nano -w /etc/collectd/collectd.conf # enable / disable / configure plugins as desired. sudo /etc/init.d/collectd restart sudo cp /usr/share/doc/collectd/examples/collection.cgi* /usr/lib/cgi-bin cd /usr/lib/cgi-bin [ -f collection.cgi.gz] && sudo gunzip collection.cgi.gz sudo chmod +x ./collection.cgi sudo apt-get install librrds-perl libconfig-general-perl libhtml-parser-perl libregexp-common-perl --yes Access http://ip.ad.re.ss/cgi-bin/collection.cgi Win!
