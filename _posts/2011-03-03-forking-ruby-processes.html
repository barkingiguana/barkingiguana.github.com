For todays adventure in Ruby, I’m going to be writing a simple daemon process. To start with this daemon won’t do anything too useful – every 1 second it’ll print the time to STDOUT. Once we have that working I’m going to use the code I created previously to check that a socket is open and increase the period to 15 seconds.
