Things I want to write or do:

  • An SRPM for ActiveMQ. I'm fed up of having to install this by hand.
  • An SRPM for DaemonTools. With sensible paths... /command does not work for me and I shouldn't need to be root to run svstat like the default instructions require.

  • Pseudo-application: expose a queue or topic via XMPP PubSub. Seems like Apache Camel can bridge between various queue implementations (thanks @jstrachan).
  • A service that searches hashtags and puts the results on the bus.
  • Application that allows people to search for NXER trains by price rather than date: When can I travel to Edinburgh and back for less than £50?
  • A PHP implementation of SMQueue.
  • Separate out the transport layer for SMQueue and provide a reference plugin for JSON.
  • Asynchronous attachment_fu. A backend that will push image resizes / thumbnailing / uploading to the blog store onto the queue.
